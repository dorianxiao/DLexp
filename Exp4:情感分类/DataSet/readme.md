# 数据集描述
```
  -train.txt   包含2W条左右中文电影评论，其中正负向评论各1W条左右。
  -validation.txt   包含6K条左右中文电影评论，其中正负向评论各3K条左右。
  -test.txt   包含360条左右中文电影评论，其中正负向评论各180条左右。
  -wiki_word2vec_50.bin   预训练词向量，中文维基百科词向量word2vec。
```
